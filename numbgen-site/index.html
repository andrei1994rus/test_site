Number generator

Input number,range and click button.

Notice: You need to input count of random numbers (left text field) and max number in range (right text field).

Count is how many random numbers you want. Count isn't able to be 0 and less and must be integer.

Max number is number to what you want to generate random number. Default max number is 3000 (if this text field is empty). Range is {0,max number}. Max number isn't able to be 0 and less and must be integer.
